# LibriSpeech
```
cd <REPOSITORY_ROOT>/egs/librispeech/common/
. ./prepare.sh <DATASET_DIR> <#SPEAKERS>
```

Then,
```
cd <REPOSITORY_ROOT>/egs/librispeech/<MODEL_NAME>/
. ./train.sh <OUTPUT_DIR>
```
for DNN training.
